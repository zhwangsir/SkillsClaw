## 📖 背景

ClawHub 是优秀的 Skills 分发平台，但存在以下限制：

- ⏳ **下载限速** - 服务器限速，安装速度不稳定
- 🌐 **网络依赖** - 国内访问经常超时或失败
- 🔄 **重复下载** - 换设备需要重新下载每个 Skill
- 📴 **离线困难** - 无网络时无法获取已安装的 Skills

**解决方案**：自建备份库，将所有 Skills 同步到 GitHub，实现永久保存、一键恢复。

---

## ✨ 特性

- 🚀 **永久保存** - 所有 Skills 永久存储在 GitHub
- ⚡ **一键恢复** - `git clone` 即可恢复全部 Skills
- 📦 **离线可用** - 本地完整副本，无需网络
- 🔐 **完全掌控** - 私有仓库，只有自己能访问
- ✏️ **自由定制** - 可随意修改 Skills 适应需求

## 🚀 快速开始

### 克隆仓库

```bash
git clone https://github.com/zhwangsir/SkillsClaw.git
```

### 恢复 Skills

**macOS / Linux:**
```bash
cp -r SkillsClaw/* ~/.openclaw/workspace/skills/
```

**Windows (PowerShell):**
```powershell
Copy-Item -Recurse -Force ".\SkillsClaw\*" "$env:USERPROFILE\.openclaw\workspace\skills\"
```

---

## 🔄 同步更新

当你添加或修改了 Skills，执行以下命令同步到 GitHub：

```bash
cd ~/.openclaw/workspace/skills

git add .
git commit -m "chore: 更新 Skills"
git push
```
